// ‡∏Å‡∏≥‡∏´‡∏ô‡∏î‡∏à‡∏∏‡∏î‡∏Å‡∏≥‡πÄ‡∏ô‡∏¥‡∏î (sites) 3 ‡∏à‡∏∏‡∏î
const sites = [
  { name: "P1", x: 0, y: 0 },

// ‡∏ü‡∏±‡∏á‡∏Å‡πå‡∏ä‡∏±‡∏ô‡∏£‡∏∞‡∏¢‡∏∞‡∏ó‡∏≤‡∏á‡∏ï‡∏≤‡∏°‡∏™‡∏π‡∏ï‡∏£‡∏Ñ‡∏ì‡∏¥‡∏ï
// d = sqrt( (x2 - x1)^2 + (y2 - y1)^2 )
function distance(x1, y1, x2, y2) {
  const dx = x2 - x1;          // ‡∏™‡πà‡∏ß‡∏ô‡∏ï‡πà‡∏≤‡∏á‡πÅ‡∏Å‡∏ô x
  const dy = y2 - y1;          // ‡∏™‡πà‡∏ß‡∏ô‡∏ï‡πà‡∏≤‡∏á‡πÅ‡∏Å‡∏ô y
  return Math.sqrt(dx*dx + dy*dy); // ‡∏ñ‡∏≠‡∏î‡∏£‡∏≤‡∏Å‡∏ó‡∏µ‡πà‡∏™‡∏≠‡∏á
}

// ‡∏à‡∏∏‡∏î‡∏ó‡∏µ‡πà‡∏≠‡∏¢‡∏≤‡∏Å‡∏£‡∏π‡πâ‡∏ß‡πà‡∏≤‡∏≠‡∏¢‡∏π‡πà‡πÉ‡∏ô‡πÄ‡∏Ç‡∏ï‡∏Ç‡∏≠‡∏á‡πÉ‡∏Ñ‡∏£

    nearestSite = site;
  }
}

‚ö° ‡πÄ‡∏ó‡∏Ñ‡∏ô‡∏¥‡∏Ñ‡πÄ‡∏û‡∏¥‡πà‡∏°‡∏õ‡∏£‡∏∞‡∏™‡∏¥‡∏ó‡∏ò‡∏¥‡∏†‡∏≤‡∏û: ‡πÄ‡∏õ‡∏£‡∏µ‡∏¢‡∏ö‡πÄ‡∏ó‡∏µ‡∏¢‡∏ö‡∏£‡∏∞‡∏¢‡∏∞‡∏Å‡∏≥‡∏•‡∏±‡∏á‡∏™‡∏≠‡∏á‡πÅ‡∏ó‡∏ô

// ‡πÄ‡∏õ‡∏£‡∏µ‡∏¢‡∏ö‡πÄ‡∏ó‡∏µ‡∏¢‡∏ö‡∏£‡∏∞‡∏¢‡∏∞‡∏Å‡∏≥‡∏•‡∏±‡∏á‡∏™‡∏≠‡∏á‡πÅ‡∏ó‡∏ô ‡∏ñ‡πâ‡∏≤‡∏à‡∏∞‡πÄ‡∏≠‡∏≤‡πÅ‡∏Ñ‡πà "‡πÉ‡∏Ñ‡∏£‡πÉ‡∏Å‡∏•‡πâ‡∏Å‡∏ß‡πà‡∏≤"

// ‡∏ü‡∏±‡∏á‡∏Å‡πå‡∏ä‡∏±‡∏ô‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ì Voronoi vertex (circumcenter) ‡∏à‡∏≤‡∏Å 3 ‡∏à‡∏∏‡∏î
function voronoiVertex(p1, p2, p3) {
  const x1 = p1.x, y1 = p1.y;
  const x2 = p2.x, y2 = p2.y;
  const x3 = p3.x, y3 = p3.y;

  // ‡∏ï‡∏±‡∏ß‡∏™‡πà‡∏ß‡∏ô D
  const D = 2 * (
    x1 * (y2 - y3) +
    x2 * (y3 - y1) +
    x3 * (y1 - y2)
  );

  if (D === 0) {
    return null; // ‡∏™‡∏≤‡∏°‡∏à‡∏∏‡∏î‡∏≠‡∏¢‡∏π‡πà‡πÄ‡∏™‡πâ‡∏ô‡∏ï‡∏£‡∏á‡πÄ‡∏î‡∏µ‡∏¢‡∏ß‡∏Å‡∏±‡∏ô
  }

  const x1sq_ysq = x1*x1 + y1*y1;
  const x2sq_ysq = x2*x2 + y2*y2;
  const x3sq_ysq = x3*x3 + y3*y3;

  const Vx = (
    x1sq_ysq * (y2 - y3) +
    x2sq_ysq * (y3 - y1) +
    x3sq_ysq * (y1 - y2)
  ) / D;

  const Vy = (
    x1sq_ysq * (x3 - x2) +
    x2sq_ysq * (x1 - x3) +
    x3sq_ysq * (x2 - x1)
  ) / D;

  return { x: Vx, y: Vy };
}
